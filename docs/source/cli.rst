==================
Command line tools
==================

.. note:: Not currently implemented.

The following command line tools are installed as part of the package

* ``mci_ipm_extract`` converts Mastercard IPM files to csv
* ``mci_ipm_param_encode`` changes the encoding of a Mastercard IPM parameter files
* ``mci_ipm_encode`` changes the encoding of a Mastercard IPM file
* ``mci_csv_to_ipm`` creates an IPM file from a csv file
